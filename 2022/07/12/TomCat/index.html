TomCat

发表于 2022-07-12 分类于 WebService

简介

Web 服务器是一个应用程序 对HTTP协议的操作进行封装，使的程序员不必直接对协议进行操作，让Web开发更加便捷

TomCat 目录结构

  • bin 可执行文件存放目录
  • conf 配置文件存放目录
  • lib tomcat依赖的jar包
  • logs 日志文件
  • temp 临时文件
  • webapps 应用发布目录
  • work 工作目录

    TomCat 基本使用

  • 配置
    • 修改启动端口号: conf/server.xml

HTTP协议默认端口号为80，如果将TomCat端口号改为80，则将来访问时将不用输入端口号

  • 启动时可能出现的问题
    • 端口号冲突 找到对应程序 将其关闭
    • 闪退 检查JAVA_HOME 环境变量是否正确配置
  • TonCat 部署项目
    • 将项目放置到webapps 目录下，即部署完成
  • 一般JavaWeb项目会被打成war包 然后将war包放到webapps目录下,TomCat会自动解压缩war文件

IDEA中创建Maven Web 项目

  • 使用骨架(项目模板)
    • 选择web项目骨架，创建项目
    • 删除pom.xml中多余的坐标
    • 补齐缺失的目录结构
      • java
      • resources
  • 不使用
    • 新建maven模板
    • 将打包方式改为war
    • 补齐缺失的目录结构
      • webapp 以及其子目录

IDEA中使用TomCat

  • pom.xml 添加TomCat插件
  • 使用Maven Helper 插件启动项目 选中项目 右键-> Run Maven –> tomcat7:run
