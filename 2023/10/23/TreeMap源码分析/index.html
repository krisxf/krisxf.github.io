TreeMap 简介

本篇使用的jdk源码版本jdk11

官方介绍(节取) :
A Red-Black tree based NavigableMap implementation. The map is sorted according to the natural ordering of its keys, or by a Comparator provided at map creation time, depending on which constructor is used.
This implementation provides guaranteed log(n) time cost for the containsKey, get, put and remove operations.
Note that this implementation is not synchronized, the map should be “wrapped” using the Collections.synchronizedSortedMap method.

TreeMap 是基于红黑树的 NavigableMap 实现。映射根据其键的自然顺序进行排序，或者由映射创建时提供的比较器进行排序，具体取决于所使用的构造函数。
此实现为 containsKey、get、put 和 remove 操作提供有保证的 log（n） 时间成本。
该类不是线程安全的,在并发环境下使用应该使用Collections.synchronizedSortedMap 方法包装
不可加入NULL值

TreeMap 继承了 NavigableMap、 Cloneable、Serializable 、AbstractMap等接口

  • NavigableMap ：TreeMap 实现了 NavigableMap 接口，它是 Map 接口的扩展，提供了导航映射的功能。NavigableMap 接口定义了一系列用于获取最小键、最大键、小于或大于给定键的子映射等方法。
  • Cloneable : 也是一个标记接口，表明它具有拷贝能力，可以进行深拷贝和浅拷贝
  • Serializable : 也是一个标记接口，表明它可以进行序列化操作，也就是可以将对象转化为字节流进行持久化存储或者在网络中传输，非常方便
  • AbstractMap : TreeMap 继承了 AbstractMap 抽象类。AbstractMap 实现了 Map 接口的大部分方法，但留下了一些方法需要具体的 Map 实现类来实现。TreeMap 通过继承 AbstractMap 来获得 Map
